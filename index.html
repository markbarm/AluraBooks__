Glossário

<head>

Tag semântica usada para incluir metadados como conjunto de caracteres, link de estilos, título que aparece na aba do navegador, etc.

<head>
  <meta charset="UTF-8">
  <title>AluraBooks</title>
  <link rel="stylesheet" href="style.css">
</head>

<header>

Tag semântica usada para implementar um cabeçalho no projeto

<header class="cabeçalho"></header>

class

Atributo de qualquer tag, usado para para criar uma classe específica.

<header class="cabeçalho"></header>

O valor do atributo, "cabeçalho", será usado como seletor, ".cabeçalho" nos arquivos css.

.cabeçalho {
  background-color: blue;
}

<span>

Tag não semântica usada para inserir um elemento, sem produzir quebra de linha, que receberá um estilo específico.

<span class="cabeçalho__menu-hamburguer"></span>

<img>

Tag semântica usada para inserir uma imagem como elemento em linha. É necessário usar os atributos "src" e "alt".

<img src="img/Favoritos.svg" alt="Meus Favoritos">

src

Atributo da tag <img> usado para indicar o caminho fonte da imagem a ser inserida.

<img src="img/Favoritos.svg" alt="Meus Favoritos">

alt

Atributo da tag <img> usado para incluir o texto alternativo da imagem a ser inserida.

<img src="img/Favoritos.svg" alt="Meus Favoritos">

<a>

Tag semântica "âncora", usada para inserir um elemento que redireciona para um link externo, quando clicado

<a href="www.alura.com.br/artigos/">Artigos alura</a>
