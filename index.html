Glossário

<header>

É uma tag semântica usada para implementar um cabeçalho no projeto

<header class="cabeçalho">

class

atributo de qualquer tag, usado para para criar uma classe específica.

<header class="cabeçalho">

O valor do atributo, "cabeçalho", será usado como seletor, ".cabeçalho" nos arquivos css.

.cabeçalho {
  background-color : blue;
}

<span>

tag não semântica usada para inserir um elemento, sem produzir quebra de linha, e que receberá um estilo específico.

<span class="cabeçalho__menu-hamburguer"></span>

<img>

É uma tag semântica usada para inserir uma imagem como elemento em linha. É necessário usar os atributos "src" e "alt".

<img src="img/Favoritos.svg" alt="Meus Favoritos">

src

atributo da tag <img> usado para indicar o caminho fonte da imagem a ser inserida.

<img src="img/Favoritos.svg" alt="Meus Favoritos">

alt

atributo da tag <img> usado para incluir o texto alternativo da imagem a ser inserida.

<img src="img/Favoritos.svg" alt="Meus Favoritos">
